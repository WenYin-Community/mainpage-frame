# 玟茵开源社区主页

玟茵开源社区的展示页，用于展示玟茵开源社区的项目、功能。

技术：Vue 3 + Vite + SCSS +  TypeScript + Element Plus



## 说明

网站虽然说没有使用到后端技术，为了代码复用及维护方便，因此网页中很多的文字和图片的链接都外置成了 JSON 放在了 data 文件夹里面。写法有点类似前后端分离技术。

同时也为了网站以后如果用到了后端，这样更方便改写。
